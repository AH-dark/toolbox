# AHdark's Toolbox

AHdark's Toolbox, provide simple services.

## Environment

### Require

* Node.js 14+

### Environment

| 环境变量     | 用途   | 默认                     |
|----------|------|------------------------|
| BASE_URL | 后端地址 | http://localhost:9000/ |

### Backend

View <https://github.com/AH-dark/toolbox-backend>

## Build

```bash
npm install --production
npm run build
```

编译后的 html 文件将被放至 `build/` 目录下
